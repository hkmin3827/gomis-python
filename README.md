# Gomis Motion Control

손 제스처만으로 Windows 데스크톱을 제어하는 로컬 실행 앱.  
키보드·마우스 없이 웹캠 앞에서 손을 움직이면 커서 이동, 클릭, 스크롤, 볼륨 조절, 창 전환이 가능합니다.

---

## 동작 원리

```
웹캠 → OpenCV(프레임) → MediaPipe(21개 손 좌표) → GestureEngine(제스처 판별) → PyAutoGUI/pycaw(OS 제어)
```

- **완전 로컬 실행** — 인터넷 연결 불필요, 원격 서버 없음
- **OS 접근 수준** — 일반 사용자 권한만 사용. 커서·클릭은 PyAutoGUI가 Windows `SendInput()` API를 통해 가상 이벤트로 전송 (드라이버·커널 접근 없음)
- **음성 인식(2단계)** — Whisper 로컬 모델, 외부 전송 없음

---

## 기술 스택

| 역할                | 기술                        |
| ------------------- | --------------------------- |
| 언어                | Python 3.10+                |
| 카메라 캡처         | OpenCV (`cv2.VideoCapture`) |
| 손 랜드마크 감지    | MediaPipe Hands (Tasks API) |
| 커서·클릭 제어      | PyAutoGUI                   |
| 볼륨 제어           | pycaw (Windows COM API)     |
| 창 전환             | pywin32 (Win32 API)         |
| UI                  | PyQt5 + pystray             |
| 음성 인식 (2단계)   | OpenAI Whisper (로컬)       |
| Claude 연동 (2단계) | Anthropic Python SDK        |

## 기능 요구사항 (명세)

### 1단계 — 핵심 제어

| 기능          | 설명                                                                              |
| ------------- | --------------------------------------------------------------------------------- |
| 커서 이동     | 검지만 편 상태에서 손 위치로 커서 이동 (웹캠 상하좌우 20% 데드존)                 |
| 클릭          | **손바닥 펴기 → 오므리기** 순서로만 발화 (ZOOM 상태 제외)                         |
| 더블클릭      | 손바닥→오므리기 동작을 0.4초 이내 2회 반복                                        |
| 스크롤        | 검지+중지 V자로 DRAG 모드 진입 → 손 위아래 이동량에 비례 스크롤 (20px 누적당 1회) |
| 줌 인         | 엄지+검지 L자 포즈 진입 후 벌리기 (Ctrl+휠, 방향 고정)                            |
| 줌 아웃       | 엄지+검지 L자 포즈 진입 후 좁히기 (Ctrl+휠, 오픈팜 후 방향 전환)                  |
| 볼륨 업       | **오른손** 샤카(엄지+새끼) → 연속 상승 (최대 3/초)                                |
| 볼륨 다운     | **왼손** 샤카(엄지+새끼) → 연속 하강 (최대 3/초)                                  |
| 창 전환(다음) | **오른손**으로 오른쪽 스와이프 (IDLE 상태, 스와이프당 1회 보장)                   |
| 창 전환(이전) | **왼손**으로 왼쪽 스와이프 (IDLE 상태, 스와이프당 1회 보장)                       |
| 제스처 종료   | 손바닥 펴기 — 손가락 끝이 PIP(중간 관절) 위까지 올라와야 인정                     |

### 2단계 — Claude 연동

- 특정 제스처 → Whisper 음성 인식 → Claude API 호출 → 응답 표시

### 3단계 — 부가 기능

- 밝기 조절, 미디어 재생/일시정지, 스크린샷, 잠금 모드, 제스처 커스터마이징 UI, 디버그 모드

## 손동작 레퍼런스

| 제스처        | 손 모양 / 동작                     | 방식                                    |
| ------------- | ---------------------------------- | --------------------------------------- |
| 커서 이동     | 검지만 펴기 + 손 이동              | 연속 (CURSOR 상태)                      |
| 클릭          | **손바닥 → 오므리기** 순서 필수    | 원샷                                    |
| 더블클릭      | 손바닥 → 오므리기 × 2 (0.4초 이내) | 원샷 × 2                                |
| 스크롤        | 검지+중지 V자 → 손 위아래 이동     | 연속 (DRAG 상태) — Y 이동량 비례 스크롤 |
| 줌 인         | 엄지+검지 L자 → 벌리기             | 연속 (ZOOM 상태) — 방향 고정            |
| 줌 아웃       | 엄지+검지 L자 → 좁히기             | 연속 (ZOOM 상태) — 오픈팜 후 전환       |
| 볼륨 업       | **오른손** 샤카 (엄지+새끼)        | 연속 (VOLUME 상태)                      |
| 볼륨 다운     | **왼손** 샤카 (엄지+새끼)          | 연속 (VOLUME 상태)                      |
| 창 전환(다음) | **오른손** + 오른쪽 스와이프       | 원샷 (IDLE, 1회 보장)                   |
| 창 전환(이전) | **왼손** + 왼쪽 스와이프           | 원샷 (IDLE, 1회 보장)                   |
| 제스처 종료   | 손바닥 펴기 (PIP 기준 엄격)        | 모든 연속 상태 종료                     |

> 레퍼런스 이미지: `docs/motion-capture-img/`

## 시스템 동작 방식

```
┌─────────────────────────────────────────────────────┐
│                    main.py (앱 시작)                  │
└──────────────────────┬──────────────────────────────┘
                       │
         ┌─────────────▼─────────────┐
         │   Camera (OpenCV)          │  ← 30fps 프레임 캡처
         └─────────────┬─────────────┘
                       │ BGR 프레임
         ┌─────────────▼─────────────┐
         │   HandTracker (MediaPipe)  │  ← 21개 랜드마크 + 손 방향(Left/Right)
         └─────────────┬─────────────┘
                       │ HandResult
         ┌─────────────▼─────────────┐
         │   GestureEngine (상태기계) │  ← 제스처 이름 반환
         │   IDLE / CURSOR / DRAG /   │
         │   ZOOM / VOLUME            │
         └─────────────┬─────────────┘
                       │ gesture name
         ┌─────────────▼─────────────┐
         │      Controllers           │
         │  cursor / scroll /         │  ← PyAutoGUI, pycaw, pywin32
         │  volume / window_switch    │
         └───────────────────────────┘
```

**제스처 상태 전이:**

```
[IDLE] ──꼬집기 (3프레임)──→ [ZOOM]    손바닥 펴기로 종료   ← zoom 우선
       ──검지만 펴기 (3f)──→ [CURSOR]  손바닥 펴기로 종료
       ──V자 (3프레임)──────→ [DRAG]    손바닥 펴기로 종료
       ──샤카 (3프레임)─────→ [VOLUME]  손바닥 펴기로 종료
       ──빠른 수평 스와이프──→ WINDOW_SWITCH (원샷, 상태 없음, 오른손↔오른쪽/왼손↔왼쪽)

[어떤 상태에서든]
       ──오므리기 (즉시)────→ CLICK / DOUBLE_CLICK (원샷, 최우선)

※ 오픈팜 후 0.5초 쿨다운 — 새 제스처 차단 (손 준비 시간)
※ 상태 진입 전 포즈를 3 프레임(~100ms) 유지해야 확정
```

## 설치 및 실행

```powershell
# 1. 가상환경 생성 및 활성화
python -m venv venv
.\venv\Scripts\Activate.ps1

# 2. 패키지 설치
pip install -r requirements.txt

# 3. 환경변수 설정
copy .env.example .env
# .env 파일에 ANTHROPIC_API_KEY 입력

# 4. 실행
python main.py

# 웹캠 + 랜드마크 테스트
python test_camera.py
```

> **주의:** 실행 전 반드시 가상환경(`venv`) 활성화 확인

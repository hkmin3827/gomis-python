포커스된 입력창에 음성으로 바로 입력할 수 있습니다.
  • 양손 박수 제스처로 음성 인식을 시작합니다
  • Whisper small 모델이 로컬에서 음성을 텍스트로 변환합니다 — 외부 서버 전송 없음
  • 커서가 깜빡이는 포커스된 텍스트 입력창에 변환된 텍스트가 자동 입력됩니다
  • 입력 후 자동 Enter가 실행되어 검색 · 전송 · 제출 동작이 수행됩니다. 자동 Enter 설정 → 자동 Enter 키 Off 설정으로 조절할 수 있습니다.
  • 기본적으로 클립보드에 복사되어 Ctrl+V로 인식된 음성 텍스트를 원하는 곳에 붙여넣기 할 수 있습니다.
  • 메모장, 브라우저 주소창, 채팅, 검색창 등 모든 입력 필드에서 동작합니다
  • 다시 양손 박수를 치면 인식이 종료됩니다
  • 최대 녹음 시간 5분(300초) — 초과 시 자동 종료 및 입력. 5분 기준 약 3,000~4,500자 분량 처리 가능
